# Coffee Page ☕

## 📌 Sobre o Projeto

O **Coffee Page** é um projeto de site estático desenvolvido para representar uma cafeteria moderna. O objetivo é apresentar informações como menu, avaliações, localização e identidade visual de forma simples, elegante e responsiva.

Este projeto foi criado como prática de desenvolvimento front-end utilizando HTML e CSS.

---

## 🚀 Funcionalidades

* Página inicial com destaque visual da marca
* Navegação por seções (Home, Sobre, Menu, Avaliações e Endereço)
* Layout organizado e responsivo
* Integração com fontes externas (Google Fonts)
* Exibição de imagens e ícones

---

## 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3
* Google Fonts

## ▶️ Como Executar o Projeto

1. Clone o repositório:

```
git clone https://github.com/seu-usuario/coffee-page.git
```

2. Acesse a pasta do projeto:

```
cd coffee-page
```

3. Abra o arquivo `index.html` no navegador.
